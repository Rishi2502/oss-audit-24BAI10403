## 📜 Project Description

This project is part of the Open Source Software course. It includes analysis of Git as an open-source software and implementation of 5 Linux shell scripts demonstrating basic Linux concepts.

## 📂 Scripts Overview

### 1. System Identity Report

Displays system details like kernel version, user, uptime, and OS information.

### 2. Package Inspector

Checks whether Git is installed and displays its details.

### 3. Disk and Permission Auditor

Analyzes important directories and shows permissions and disk usage.

### 4. Log File Analyzer

Reads log files and counts occurrences of specific keywords.

### 5. Manifesto Generator

Creates a personalized open-source philosophy statement.

## ▶️ How to Run Scripts

```bash
chmod +x script1_system_info.sh
./script1_system_info.sh
```

Repeat for all scripts.

For log analyzer:

```bash
./script4_log_analyzer.sh /var/log/syslog error
```

## ⚙️ Requirements

* Linux OS (Ubuntu recommended)
* Bash shell

All scripts are tested on a Linux environment and demonstrate basic shell scripting concepts like loops, conditions, variables, and file handling.
